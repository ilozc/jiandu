使用手册 · 笺读

一份速查手册:第一次用从「三步上手」开始,之后随时点右上角 ? 回来查。

三步上手
1导入 PDF:点「+ 导入」或把 PDF 拖进窗口;可多篇,每篇成为左侧一章。
2提你的问题(可选,在「更多 ▾ → 我的问题」):写下最想弄清的几点(最多 5 个),AI 逐条作答、旁注按问题分色。
3点「AI 精读」:自动逐页生成旁注 + 顶部全文总览;过程中可切到别的论文,结果仍写回原文。
怎么读批注

每条旁注落在页边、用曲线连回原文,并标一个“层”;设过问题的话还会按问题分色。

结论针对你某个问题给出的答案或判断。
机制最值得看:怎么做到的——动了什么结构/操作、关键参数、为什么有效。
证据支撑结论的实测数据 / 指标(含数值与单位)。
局限代价、适用边界、存疑或前后不一致之处。

顶部「师兄带读」卡是全文总览:做了什么、机制链、对你的落点、仿真要点、按问题的「答卷」。没设问题也能用,只是不分色。

精读模型
只存在本机浏览器(明文 localStorage),不会上传到别处。
精读推荐 deepseek-v4-pro / kimi-k2.6 / claude-opus-4-8。可填任意模型名。
划词翻译
不勾选则翻译沿用上面精读的供应商与 Key,只换下面的「翻译模型」。
建议 deepseek-v4-flash / kimi-k2.5 / claude-haiku-4-5。留空则与精读同模型。
翻译已默认采用 en-zh-translation-polish(叶子南《高级英汉翻译理论与实践》方法:按文本类型调归化档位、去翻译腔、意合优先、准确为红线);这里填的会附加在其后。
图表精读(看图,需视觉模型)
默认不勾:图表精读是工具栏上独立的「图表精读」按钮,想看图时单独点(可单独重跑、不重复花文字 token)。勾上则每次 AI 精读完自动顺带看一遍图。两种都需要下面配好支持看图的模型。
不勾选则沿用上面精读的供应商与 Key(精读若用 Claude 等能看图的模型,直接就能用)。
需支持看图(多模态)。推荐 Claude(浏览器直连不踩 CORS)。留空则沿用上面的精读模型名。
精读 Skill 与组内偏好
已载入的多个 skill 会全部拼进 AI 系统提示;下面文本作为额外补充。
开启且填写后,会把这段偏好附加到精读与总览的指令里。留空则不影响。内容只存在你本机浏览器。
课题组 / 个人配置
把"研究偏好 + 广场关键词 + 自定义标签"打包成一个小配置文件,导入即生效(写进本机)。换电脑或发给同组同学时导一份即可,不必每次手填;导入不会动你的文献与批注。
数据备份 / 迁移
把全部文献 PDF + 批注 + 标签 + 文件夹 + 设置打包成一个 .json 文件,存到你自己的本地文件夹或网盘;换电脑时点「导入整库」即可一键还原。这样数据就不只绑在某个浏览器里。
